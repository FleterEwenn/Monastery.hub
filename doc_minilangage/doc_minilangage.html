Monastery Language

Initialisation ou modification de variable

nindroide nom = valeur

  • nindroide est la fonction permettant de modifier ou creer une variable
  • name est le nom de la variable à modifier ou à créer
  • valeur est la valeur attribuée à la variable, les types des variables correspondent aux types de donnée de python

Affichage dans la console

spinjutzu name/valeur

  • spinjutzu est la fonction permettant d'afficher quelque chose dans la variable
  • name/valeur est ce qu'il faut afficher dans la console, cela put donc être soit le nom d'une variable soit une valeur quelconque respectant les types de donnée de python

Instruction conditionelle / instruction si-alors

wu condition {
code à executer si la condition est vraie
}

  • wu est le mot clé correspondant au si (soit if dans la plupart des langage)
  • conditon est donc une condition retournant un booléen, elle doit être vraie pour éxécuter le code
  • {} délimite le code à éxécuter, l'indentation n'est donc pas nécessaire
