# Monastery.hub

## Pour commencer

Avant de lancer le projet, il faut d'abord installer les bibliothèques nécessaires a l'éxécution du projet :
```
pip install -r requirements.txt
```

## Démarrage

Maintenant, le demarrage du projet est possible.
Pour cela, il faut executer le fichier main.py :
```
python main.py
```
